# Assets Directory

This directory contains static assets for the Coercive Control Analyzer project.

## Logo Files

The `logos/` directory contains the official branding for the project:

### Required Logo Files

Place the following logo files in the `logos/` directory:

1. **logo.png** - Main logo in PNG format (recommended: 512x512px or higher)
2. **logo-light.png** - Logo optimized for light backgrounds
3. **logo-dark.png** - Logo optimized for dark backgrounds
4. **logo-small.png** - Small version for headers (recommended: 200x200px)
5. **favicon.ico** - Favicon for web applications (16x16, 32x32, 48x48)
6. **logo.svg** - Scalable vector version (optional but recommended)

### Logo Specifications

**Brand Identity:**
- Name: "Coercive Control Analyzer"
- Tagline: "Personal Development"
- Design: Circular design with decorative elements

**Technical Requirements:**
- Maintain quality at various sizes
- Optimize file sizes for web use
- Professional appearance suitable for forensic/legal documentation
- Support for light/dark mode contexts

### Usage

The logo is automatically integrated into:
- README.md header
- HTML report headers
- Generated forensic documentation
- Court-ready reports

### File Organization

```
assets/
├── logos/
│   ├── logo.png           # Main logo
│   ├── logo-light.png     # Light background variant
│   ├── logo-dark.png      # Dark background variant
│   ├── logo-small.png     # Small header version
│   ├── favicon.ico        # Web favicon
│   └── logo.svg           # Vector version (optional)
└── README.md              # This file
```

### Notes

- Logo files should be committed to version control
- Use PNG format for bitmap images with transparency
- SVG format is preferred for scalability
- Maintain consistent branding across all materials
